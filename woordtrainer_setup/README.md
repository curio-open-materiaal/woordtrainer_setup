# Woordtrainer setup (installatie / update)

Zie ook de [README in de repo-root](../README.md) voor de volledige documentatie.

## Snel na een nieuwe Git-deploy

Na `git pull` of deploy: log in als admin, open `/woordtrainer_setup/index.php`, klik **Installeer Woordtrainer**. Reset is niet nodig voor normale updates.

## Release

Versie staat op de installatiepagina en in `config.php` → `release`.

## ElevenLabs

API-sleutel instellen op de installatiepagina → opgeslagen in `storage/settings.json`.
